# Homelab Data Platform

Welcome to the homelab data platform documentation.

## Overview

This homelab runs a complete data platform on Kubernetes, managed with GitOps principles using Flux.

## Key Components

- **Infrastructure**: Kubernetes cluster with Flux CD for GitOps
- **Storage**: Rook/Ceph for distributed storage
- **Data Platform**: Catalog, orchestration, and processing tools
- **Applications**: VS Code Server, Keycloak, and more
- **Monitoring**: Observability stack for the entire platform
